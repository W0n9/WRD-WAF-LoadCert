## 配置说明
使用`config.json`配置文件，配置说明如下：
| 参数名称 | 参数说明 |
| -------- | -------- |
| `user` | 用户名 |
| `password` | 密码 |
| `baseURL` | RESTful接口地址，含Scheme和端口 |
| `certificate_path` | 证书路径 |
| `key_path` | 私钥路径 |
